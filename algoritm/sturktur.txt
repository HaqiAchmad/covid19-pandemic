com = super package

com.main.Main 
	  - main()
notes : class Main adalah class yang pertama kali dijalankan saat program dijalankan
      : class ini hanya berisi satu method yaitu main() yang digunakan untuk membuka aplikasi

com.database.Database
	      + Connection conn
	      + Statement stat
	      + ResultSet res
	      + final String DRIVER = driver yg digunakan untuk terhubung ke database
	      + 	     URL = link ke database
    	      + 	     USER = nama username 
	      + 	     PASS = password
	      - void startConnection()
	      - void closeConnection()
	      - Object[][] getData(final String sql)
	      - void readData(final String sql)
	      - boolean updates(final String sql) 
notes : class ini digunakan aplikasi untuk berkomunikasi dengan database
	seperti membuat koneksi ke database, memutus koneksi ke database
	melakukan pengeditan data yang ada di dalam database

	      + Connection conn

	      + ResultSet res

	      - boolean login(fString user, fString password)
	      - boolean changePassword(fString user, fString newPassword)
	      - boolean logout()
	      - boolean isLogin() // untuk mengecek apkah user sdh login atau blm
	      - boolean deleteAccount(fString user)
	      - boolean isValidUsername(fString username)
	      - boolean isValidNamalengkap(fString namalengkap)
	      - boolean isValidEmail(fString email)
	      - boolean isValidPassword(fString password)
notes : class ini digunakan untuk segala sesuatu yang berhubungan dengan account
